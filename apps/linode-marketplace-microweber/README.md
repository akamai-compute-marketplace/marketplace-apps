# Microweber Quick Deploy App for Akamai Cloud Compute

(README stub — written fully at the `/app-pr` phase.)

Deploys [Microweber](https://microweber.org) v2.0.20 — a drag-and-drop CMS and e-commerce
website builder built on Laravel — on Ubuntu 24.04 with Apache, PHP 8.3, and MariaDB.
The browser setup wizard is eliminated via the upstream CLI installer; the install artifact
is version-pinned and sha256-verified; HTTPS is configured automatically via Let's Encrypt.
